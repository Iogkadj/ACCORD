# Boeing 737 MAX Coordinate Viewer - Android App
    
    This is a hybrid Android app built with Capacitor that visualizes Boeing 737-8 and 737-9 MAX aircraft coordinates in 3D space.
    
    ## Features
    - Enter Station (STA), Waterline (WL), and Buttock Line (BL) coordinates
    - Visual 3D representation of coordinate location
    - Toggle between 737-8 and 737-9 MAX models
    - Interactive 3D view with rotation and zoom
    
    ## Installation
    
    See **SETUP_INSTRUCTIONS.md** for detailed setup steps.
    
    ### Quick Start
    ```bash
    npm install
    npx cap add android
    npx cap sync
    npx cap open android
    ```
    
    Then build and run from Android Studio!
    
    ## System Requirements
    - Node.js 14+
    - Android Studio
    - JDK 11+
    - Android device with API 22+ (Android 5.1+)
